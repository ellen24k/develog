# 코드블록

	Tab : 한 줄 단위 코드블록

혹은

	```java
	public class Main {
		public static void main(String[] args) {
			System.out.println("hello world");
    	}
	}
	```

	public static void main(String[] args) {
		System.out.println("hello world");
   	}

# 이미지
이미지 파일을 업로드 시

	![](이미지 링크)
    
코드가 추가된다.
이 부분을

	<img src="이미지 링크">

와 같이 바꿔줄 수 있다.

	<img src="이미지 링크" width="50%" height="300px">
와 같이 
`html의 img 태그`를 이용하면 사이즈 등을 조정할 수 있다.
